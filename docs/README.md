# Monte Athos — Pagina de Plantas

Pagina interativa para o cliente escolher unidade no **Residencial Monte Athos - Torre B**.

## Como funciona
- Lista todas as 60 unidades **DISPONIVEIS** da Torre B (4o ao 18o andar).
- Filtros: Coluna, Andar, Tipologia, Posicao Solar e Faixa de preco.
- Posicao solar: colunas pares (02, 04, 06, 08, 10, 12, 14) recebem sol da manha; impares (01, 03, 05, 07, 09, 11, 13), sol da tarde.
- Botao "Quero esta unidade" abre o WhatsApp do corretor (ou padrao Oceano).
- Sistema de corretor via URL `?c=david` (ver `data/corretores.json`).
